# TSC Raspberry Pi

Simplified version of TSC (The Source Code) for Raspberry Pi display.

## Files

- `pi-version.html` - Main interface with Hebrew support and dark theme
- `pi-tsc.js` - Core TSC card generation engine (8-parameter system)
- `elements-data.js` - Spiritual elements data and reading templates

## Installation

```bash
git clone https://github.com/icooon/TSC_pi4.git
cd TSC_pi4
```

## Usage

```bash
# Open in browser
chromium-browser pi-version.html

# Kiosk mode for exhibition
chromium-browser pi-version.html --kiosk
```

## Controls

- **Click button** "צור קלף" to generate new cards
- **Press SPACEBAR** to generate new cards
- Hebrew readings appear on the right side
